Elibrary

eLibrary is a JavaScript library for highlighting HTML elements.

Download this project as a .zip file Download this project as a tar.gz file

eLibrary is pure FP, provides both OOP-style and procedural-style API which offers some util tools for HTML elements highlighting. For more information, see README(https://github.com/bombless/eLibrary/blob/master/README.md).
